How this works

Every household task has three parts. The framework is called CPE. The goal is for one person to hold all three on a card. Mental load lives mostly in the first two.

C

Conception

Noticing something needs to happen. The most invisible work.

P

Planning

Figuring out how and when. Scheduling, deciding, organizing.

E

Execution

Actually doing the thing. Often the only piece thanked.

The Dashboard next shows what I (Mike) think we have today — based on what I see. Some I'm guessing on, some I genuinely don't know. Those are flagged discuss or open.

From there we walk through together. Move what doesn't fit. Reassign. Set the minimum standard so the holder is trusted to do it their way. We don't have to finish today.

        Family code (live sync)

        Both you and Jess must enter the same code. Edits on one device appear on the other within ~1 second. The default is fine for the two of you.

        Tip: text Jess this URL plus the family code so she can join.

        📅 Outlook calendar feeds

        Paste each of your Outlook "Publish a calendar" ICS URLs. Events from the next 7 days appear on the Dashboard's "This week's calendar" panel, color-coded by owner.

        How to get an ICS URL from Outlook
        1. Open outlook.office.com in a browser, sign in.
        2. Click the gear ⚙ → View all Outlook settings at the bottom.
        3. Calendar → Shared calendars → Publish a calendar.
        4. Pick your Calendar, set permission to "Can view all details."
        5. Click Publish. Two links appear — copy the ICS link (ends in .ics).
        6. Paste below.

        If "Publish a calendar" is missing, your IT may have disabled it. Tell me and we'll switch approaches.

        Privacy note: anyone with the URL can read your calendar. URLs sync via the family code — keep that code private.

        Filter what shows up

        Most work calendar entries are noise. Use these rules so the "This week's calendar" panel only shows events that need coordination.

        When ON, only events that match a keyword OR are on a weekend OR before/after work hours OR all-day will show. When OFF, all events show except those in "Always hide."
